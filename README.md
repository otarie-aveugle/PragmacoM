## Project Setup

### Clone the project

```sh
git clone https://github.com/otarie-aveugle/PragmacoM.git
cd PragmacoM
pnpm install
```

### Configure the .env file

```sh
mv .env.exemple .env
```

And update the `APPWRITE_PROJECT_ID`

### Run the project

```sh
pnpm dev
```

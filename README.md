## Project Setup

### Clone the project

```sh
git clone https://github.com/otarie-aveugle/PragmacoM.git
cd PragmacoM
pnpm install
```

### Configure the .env file

```sh
mv .env.exemple .env
```

Edit the `.env` file and fill in the environment variables.

### Run the project

```sh
pnpm dev
```
